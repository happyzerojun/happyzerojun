# 💻 반갑습니다, 백엔드 개발자 나영준입니다.

> **기획부터 배포까지 완주하는, 광주일보 보도로 검증된 실행력을 갖춘 백엔드 개발자입니다.**

---

### 🚀 About Me
- 💡 **검증된 실행력**: 대학가 고물가 속 학우들의 식비 부담을 줄이기 위해 기획·개발한 '가성비 맛집 지도'가 큰 호응을 얻으며 **광주일보에 단독 인터뷰로 보도**되었습니다.
- 🛠️ **탄탄한 백엔드 기초**: 정보처리기사 및 SQLD 자격을 기반으로 정규화된 데이터 모델링(ERD)과 쿼리 최적화가 가능하며, NCA 자격을 통해 클라우드 인프라에 대한 이해도를 보유하고 있습니다.
- 🤝 **소통 중심의 리더십**: 컴퓨터공학과 학생회장으로서 900명 규모의 대규모 행사를 총괄하며, 협업을 매끄럽게 이끄는 안정적인 커뮤니케이션 역량을 증명했습니다.

---
